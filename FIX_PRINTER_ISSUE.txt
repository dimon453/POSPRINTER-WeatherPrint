ЕСЛИ ВСЕ ЕЩЕ НЕ РАБОТАЕТ:
═══════════════════════════════════════════════════════════════════

1. Запусти диагностику:
   python test_printer.py

2. Проверь что "80mm printer" действительно существует в Windows:
   Параметры → Устройства → Принтеры и сканеры

3. Попробуй напечатать обычный документ из Notepad на этот принтер

4. Если win32print все еще не работает, есть два варианта:

   ВАРИАНТ A: Используй Print to File (откроется диалог сохранения)
   ─────────────────────────────────────────────────────────────
   Измени в config.json:
   "printer_name": "Print to File (PDF)"
   
   ВАРИАНТ B: Используй cmd команду lpstat (если установлен)
   ────────────────────────────────────────────────────────
   Выполни в CMD:
   lpstat -p -d
   (Должно показать список принтеров)

ДЕБАГ ИНФОРМАЦИЯ:
═══════════════════════════════════════════════════════════════════

Если печать не работает, запусти это в Python REPL:

python
>>> import win32print
>>> printers = [p[2] for p in win32print.EnumPrinters(2)]
>>> for p in printers:
...     print(p)

Должен вывести "80mm printer"

Потом попробуй печать:
>>> hprinter = win32print.OpenPrinter("80mm printer")
>>> print("✓ Принтер открыт успешно")
