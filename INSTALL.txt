🚀 НАЧАЛО РАБОТЫ (3 ШАГА):
═══════════════════════════════════════════════════════════════════

ШАГ 1️⃣ - УСТАНОВКА (2 минуты)

  Откройте PowerShell/CMD в папке проекта и выполните:

  pip install -r requirements.txt

  Если есть ошибка с pywin32:
  python -m pip install pywin32
  python -m pywin32_postinstall -install

ШАГ 2️⃣ - ПРОВЕРКА (1 минута)

  python test_printer.py

  Этот скрипт проверит:
  ✓ Наличие принтера
  ✓ Интернет соединение
  ✓ Установку модулей
  ✓ Конфигурацию

ШАГ 3️⃣ - ЗАПУСК (30 секунд)

  python weather_printer_app.py

  Потом открыть браузер:
  http://localhost:5000

⚙️ ПЕРВИЧНАЯ НАСТРОЙКА (в веб-интерфейсе):
═══════════════════════════════════════════════════════════════════

1. API КЛЮЧ OpenWeatherMap
   → Получить на https://openweathermap.org/api
   → Вставить в поле "API ключ"

2. ГОРОД
   → Оставить "Chișinău" или изменить
   → Используй английское название

3. ПРИНТЕР
   → Нажать "🔄 Обновить"
   → Выбрать "POS895UE" или твой принтер

4. ШИРИНА ЛЕНТЫ
   → Оставить 60мм (стандарт для POS895UE)

5. РАСПИСАНИЕ (опционально)
   → Включить если нужна автоматическая печать
   → Выбрать время и дни недели

6. СОХРАНИТЬ И ЗАПУСТИТЬ
   → Нажать зеленую кнопку "💾 Сохранить и Запустить"

🔧 СТРУКТУРА РАБОТЫ:
═══════════════════════════════════════════════════════════════════

ДВУХЧАСТНАЯ АРХИТЕКТУРА:

1️⃣ GUI (weather_printer_app.py)
   • Запускается на локальной машине
   • Веб-интерфейс на http://localhost:5000
   • Управление конфигурацией
   • Интерфейс для пользователя

2️⃣ ФОНОВЫЙ СЕРВИС (weather_bot.py)
   • Запускается отдельным процессом
   • REST API на http://localhost:5001
   • Работает по расписанию (APScheduler)
   • Выполняет печать на принтер
   • Получает команды от GUI

КОММУНИКАЦИЯ:
   GUI ←(REST API)→ Фоновый сервис ←→ Принтер

⚡ СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ:
═══════════════════════════════════════════════════════════════════

ВАРИАНТ 1: РУЧНАЯ ПЕЧАТЬ
   • Запустить приложение
   • Открыть http://localhost:5000
   • Нажимать кнопку "Печать Сейчас" по необходимости

ВАРИАНТ 2: АВТОМАТИЧЕСКАЯ ПЕЧАТЬ ПО РАСПИСАНИЮ
   • Включить "Включить автоматическую печать"
   • Установить время (например, 09:00)
   • Выбрать дни (пн-пт или все дни)
   • Оставить приложение работающим в фоне
   • Каждый день в указанное время будет печать

ВАРИАНТ 3: АВТОЗАПУСК ПРИ ВКЛЮЧЕНИИ ПК
   • Запустить setup_autorun.bat
   • Приложение будет запускаться автоматически
   • Будет печатать по установленному расписанию
